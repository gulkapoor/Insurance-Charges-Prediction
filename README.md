# Insurance Charges Analysis and Prediction

## Project Overview
This project focuses on analyzing medical insurance data using Python. The project includes complete data preprocessing, exploratory data analysis (EDA), feature engineering, statistical analysis, and machine learning preparation techniques.

The goal of the project is to understand the factors affecting insurance charges and prepare the dataset for predictive modeling.

---

## Dataset Features
The dataset contains the following information:

- Age
- Gender
- BMI
- Number of Children
- Smoking Status
- Region
- Insurance Charges

## Project Workflow

### 1. Data Cleaning
- Checked for missing values
- Verified data consistency
- Handled categorical variables
- Removed unnecessary issues from the dataset

---

### 2. Exploratory Data Analysis (EDA)
Performed visual and statistical analysis using:
- Count plots
- Box plots
- Histograms
- Heatmaps
- Correlation analysis

EDA helped identify patterns, outliers, and relationships between variables.

---

### 3. Data Preprocessing
- Encoded categorical variables
- Prepared numerical features
- Structured data for analysis and machine learning tasks

---

### 4. Feature Engineering and Feature Extraction
Created and transformed features to improve data understanding and model preparation.

Examples include:
- BMI categories
- Encoded region and gender variables
- Smoker classification features

---

### 5. Pearson Correlation Analysis
Used Pearson Correlation to measure linear relationships between numerical variables.

This helped identify:
- Strong positive correlations
- Negative correlations
- Important features affecting insurance charges

---

### 6. Chi-Square Test
Performed Chi-Square testing to analyze relationships between categorical variables.

The test helped determine whether categorical features were significantly associated with target-related variables.

---

## Key Insights
- Smoking status strongly impacts insurance charges.
- Higher BMI is associated with increased medical expenses.
- Age shows a positive relationship with insurance costs.
- Some categorical variables show statistical association through Chi-Square testing.
